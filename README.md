# 💖 Baby Girl AI Chatbot

A cute AI chatbot built using React, Vite, and OpenRouter.

## Features
- 🤖 Real AI Responses
- 💕 Cute Chat Interface
- ⚡ Fast React + Vite
- 🌐 OpenRouter API Integration

## Technologies
- React
- Vite
- JavaScript
- CSS
- OpenRouter API

## Setup

npm install

Create a .env file:

VITE_OPENROUTER_API_KEY=your_api_key

Run:

npm run dev
